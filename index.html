Save Nautica The Totem of Destiny

Save Nautica is a game where you control Trashbot, a robot whose mission is to collect every trash dumped into the ocean. But beware, Trashbot got a limited amount of time where he can be underwater. This game was made for the Ludus Octopus Jam 2023. The latter had for theme The Protection of the oceans. So the game had to be about sea fauna and its environment. (Also, the game at its current state is exclusively in french) You can check the Itch page for Ludus Academie. And also the website for Octop'Us.

Save Nautica est un jeu dans lequel vous contrôlez Trashbot, un robot dont la mission est de collecter tous les déchets jetés dans l'océan. Mais attention, Trashbot a un temps limité où il peut être sous l'eau. Ce jeu a été réalisé pour le Ludus Octopus Jam 2023. Ce dernier avait pour thème La Protection des océans. Le jeu devait donc porter sur la faune marine et son environnement. Vous pouvez consulter la page Itch pour Ludus Academie. Et aussi le site d'Octop'Us.

The Totem of Destiny is a game in which you play a secret agent who tries to infiltrate a base full of capybaras in order to recover the Golden Totem, an artifact that holds a mysterious power. This game was made as part of a group project for Ludus Académie. (Also, the game at its current state is exclusively in french)

The Totem of Destiny est un jeu, on incarne un agent secret qui essaie de s'infiltrer dans une base remplie de capybaras dans le but de récupérer le Totem d'or, un artefact qui détiendrait un mystérieux pouvoir. Ce jeu a été réalisé dans le cadre d'un projet de groupe pour Ludus Académie.

The Totem of Destiny ist ein Spiel, in dem Sie einen Geheimagenten spielen, der versucht, eine Basis voller Capybaras zu infiltrieren, um das Goldene Totem zu bergen, ein Artefakt, das eine mysteriöse Macht besitzt. Dieses Spiel entstand im Rahmen eines Gruppenprojekts für Ludus Académie. (Außerdem ist das Spiel in seinem aktuellen Zustand ausschließlich auf Französisch)
