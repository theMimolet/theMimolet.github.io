Save Nautica

Save Nautica is a game where you control Trashbot, a robot whose mission is to collect all trash dumped into the ocean. But beware, Trashbot got a limited amount of time where he can be underwater.

Save Nautica est un jeu dans lequel vous contrôlez Trashbot, un robot dont la mission est de collecter tous les déchets jetés dans l'océan. Mais attention, Trashbot a un temps limité où il peut être sous l'eau. Ce jeu a été réalisé pour le Ludus Octopus Jam 2023. Ce dernier avait pour thème La Protection des océans. Le jeu devait donc porter sur la faune marine et son environnement.
